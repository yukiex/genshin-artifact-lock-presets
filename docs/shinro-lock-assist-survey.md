## 2. Custom Preset 調査
- `setting1` / `setting2` は厳しさではなく**異なるキャラクタータイプ**を表す。攻略サイト・ビルドガイドなど、根拠となる URL とポイントを以下に整理する。

| Preset | Archetype / 対象キャラ | 参考URL・メモ |
| --- | --- | --- |
| setting1 | ゼロエネルギー前提の氷アタッカー（スカーク、マーヴィカ） | Game8 は「元素エネルギーを0の状態で維持できる氷キャラがいない」としつつマーヴィカだけが代替エネルギーを持つと解説 ([game8.jp/genshin/676658](https://game8.jp/genshin/676658#hl_3))。Gamerch も「スカークのモチーフ聖遺物では？」と記載し、将来的に通常/爆発を交互に撃つアタッカー向けと示唆 ([gamerch.com/genshinwiki/910744](https://gamerch.com/genshinwiki/910744))。→ ER0運用を想定し、会心2種＋攻撃%を優先。 |
| setting2 | 2セット採用の氷メインDPS（綾華、甘雨、ロサリアなど） | Game8 の 2セット解説部分で「シンプルな氷ダメUPなので氷キャラと相性が良い」と明記 ([game8.jp/genshin/676658#hl_1](https://game8.jp/genshin/676658#hl_1))。4セット条件を満たせない汎用アタッカーは、氷ダメ+15%を別セットと混ぜる構成が推奨されている。→ メイン/サブは会心2本＋攻撃%、サポ用途ならERをサブで拾う想定。 |

## 3. 作業ログ
- リント: `yamllint presets docs`
- スロット確認: `yq '.preset.slots | keys' presets/shinro/*.yml`
- 必須サブステ確認: `yq '.preset.slots[] | select(.substats_required_min == null)' presets/shinro/*.yml`
